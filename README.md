# React + Vite Social Media Clone
Full-Stack Social Media Application

Description: A real-time social platform featuring authentication, feed updates, and interaction, built on a serverless architecture.

    Tech Stack: React, Firebase (Auth, Firestore), React Hot Toast.

    Key Features:

        Real-Time Sync: Leveraged Firestore's onSnapshot listeners to push updates (new posts, likes) to connected clients instantly without page refreshing.

        Google Authentication: Integrated Firebase Auth for secure login and session management.

        Serverless Image Handling: Implemented a custom Base64 encoding solution to store images directly in the database, bypassing the need for external storage buckets.

        Social Interactions: Full CRUD capabilities for Posts, plus logic for Liking (toggle state) and Commenting.
